# NTP-Clock
ESP-8266 Based NTP Clock with Temperature and Humidity using DHT11

-- Uses a single momentary push button to scroll through menu -- 
-- Connects to local WiFi to fetch Network Time --
-- 3 Options --
1. Time
2. Temperature & Humidity from DHT11
3. Carbon Monoxide and Smoke from MQ2
-- Displays the above data on a 16x2 LCD display
-- Powered by ESP8266 --
